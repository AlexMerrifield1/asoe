### Phase 2: Backend AI Agent Pipeline

**Agent 1: Web Scraper Agent**
- Input: Company website URL
- Process:
  - Use WebFetch or axios to retrieve HTML
  - Parse with cheerio to extract:
    - Company description
    - Product/service offerings
    - Recent news/announcements
    - Technology indicators
  - Structure data as JSON
- Output: `companyProfile` object

**Agent 2: Salesforce Enrichment Agent**
- Input: Company name
- Process:
  - Query Salesforce API using jsforce
  - Retrieve:
    - Past opportunities (won/lost)
    - Engagement history
    - Health scores
    - Previous project details
  - Only runs for "Closed Lost" and "Expansion" workflows
- Output: `salesforceData` object

**Agent 3: Research Agent**
- Input: `companyProfile` + `salesforceData` + workflow type
- Process:
  - Use Claude Sonnet 4.5 to analyze combined data
  - Identify key talking points relevant to workflow type
  - Map company situation to SOLVD frameworks:
    - Net New: Challenge business assumptions (e.g., "Hiring Internal is Safer")
    - Closed Lost: Find "Black Swan" re-engagement trigger
    - Expansion: Identify new pain points from company evolution
- Output: `researchFindings` object with structured insights

**Agent 4: Fact-Check Agent**
- Input: `researchFindings`
- Process:
  - Use Claude Sonnet 4.5 to validate each claim
  - Use WebSearch to verify external facts
  - Assign confidence scores (High/Medium/Low)
  - Flag statements that need manual review
- Output: `validatedFacts` object with confidence scores

**Agent 5-7: Parallel Content Generators**

**Email Generator:**
- Input: `validatedFacts` + workflow type + user's example emails
- Process:
  - Use Claude Sonnet 4.5 with custom prompt template
  - Generate subject line and body
  - Reference specific validated facts
  - Follow user's tone/style from examples
- Output: Email draft

**Loom Script Generator:**
- Input: `validatedFacts` + workflow type + user's example scripts
- Process:
  - Use Claude Sonnet 4.5 with timestamped script template
  - Structure as 90-second video outline
  - Include camera on/off cues
  - Reference specific slides
- Output: Loom script

**Slide Content Generator:**
- Input: `validatedFacts` + workflow type + user's slide deck PDFs
- Process:
  - Use Claude Sonnet 4.5 to generate slide content
  - Follow "Problem → Reality → Shift" framework
  - Extract style/structure from user's PDFs
  - Generate bullet points ready to paste
- Output: Slide content (structured JSON)

**Orchestrator Logic:**
```javascript
async function generateOutreachAssets(formData, workflowType) {
  // Step 1: Scrape website
  const companyProfile = await webScraperAgent.scrape(formData.website);

  // Step 2: Enrich with Salesforce (if applicable)
  let salesforceData = null;
  if (workflowType === 'closedlost' || workflowType === 'expansion') {
    salesforceData = await salesforceEnricher.fetch(formData.companyName);
  }

  // Step 3: Research & analysis
  const researchFindings = await researchAgent.analyze({
    companyProfile,
    salesforceData,
    workflowType,
    formData
  });

  // Step 4: Fact-check
  const validatedFacts = await factCheckAgent.verify(researchFindings);

  // Step 5: Generate content in parallel
  const [email, loomScript, slideContent] = await Promise.all([
    emailGenerator.generate(validatedFacts, workflowType),
    loomGenerator.generate(validatedFacts, workflowType),
    slideGenerator.generate(validatedFacts, workflowType)
  ]);

  // Step 6: Return complete asset package
  return {
    email,
    loomScript,
    slideContent,
    metadata: {
      companyName: formData.companyName,
      workflowType,
      generatedAt: new Date().toISOString(),
      confidenceScores: validatedFacts.scores
    }
  };
}
```

async function generateOutreachAssets(formData, workflowType) {
  // Step 1: Scrape website
  const companyProfile = await webScraperAgent.scrape(formData.website);

  // Step 2: Enrich with Salesforce (if applicable)
  let salesforceData = null;
  if (workflowType === 'closedlost' || workflowType === 'expansion') {
    salesforceData = await salesforceEnricher.fetch(formData.companyName);
  }

  // Step 3: Research & analysis
  const researchFindings = await researchAgent.analyze({
    companyProfile,
    salesforceData,
    workflowType,
    formData
  });

  // Step 4: Fact-check
  const validatedFacts = await factCheckAgent.verify(researchFindings);

  // Step 5: Generate content in parallel
  const [email, loomScript, slideContent] = await Promise.all([
    emailGenerator.generate(validatedFacts, workflowType),
    loomGenerator.generate(validatedFacts, workflowType),
    slideGenerator.generate(validatedFacts, workflowType)
  ]);

  // Step 6: Return complete asset package
  return {
    email,
    loomScript,
    slideContent,
    metadata: {
      companyName: formData.companyName,
      workflowType,
      generatedAt: new Date().toISOString(),
      confidenceScores: validatedFacts.scores
    }
  };
}
